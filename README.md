# Natural Bless - The Coffee Expedition

A premium, immersive web experience dedicated to the journey from cup to bean.

## Project Structure

- `public/`: The root directory for the web server.
  - `assets/`: Images and static media.
  - `index.html`: Main entry point.
  - `style.css`: Modern styling with glassmorphism and cinzel typography.
  - `script.js`: Interactive elements and booking logic.
- `package.json`: Project dependencies and scripts.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local server:
   ```bash
   npm start
   ```

## Key Features

- **Modern Aesthetics**: Sleek dark mode with high-contrast accents.
- **Responsive Design**: Optimized for all devices.
- **SEO Optimized**: Includes semantic HTML, meta tags, robots.txt, and sitemap.
- **Interactive Experience**: Scroll animations and dynamic booking forms.
